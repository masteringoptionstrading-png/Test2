# Trading Dashboard

This package is ready for GitHub Pages.

## Files
- index.html
- css/styles.css
- js/app.js
- data/sample.csv

## How to use
1. Upload all files to a GitHub repo.
2. Enable GitHub Pages on the repo.
3. Open the public URL.
4. Click **Upload CSV** to load your own trade history.

## Expected CSV columns
Date, Trade, Size, Entry, Avg Exit, P/L %, P/L in $

## Notes
- Trade history sorts in descending order.
- Charts, calendar, KPIs, and widget refresh automatically after upload.
